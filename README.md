# Impy-D - Graphical Client for [MPD](https://www.musicpd.org/)

![Impy-D interface screenshot](/screenshots/Screen4Short.webp)

Impy-D aims to be a snappy, utilitarian client with a highly customizable music library view and support for whacky visualizations.

It uses [ImGui](https://github.com/ocornut/imgui) running on [GLFW](https://www.glfw.org/).
